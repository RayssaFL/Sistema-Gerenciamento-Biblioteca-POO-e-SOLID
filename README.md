<p align="justify">
Este diagrama de classes representa a estrutura de um sistema de gerenciamento de biblioteca,
desenvolvido com base nos princípios da <b>Programação Orientada a Objetos</b> e nas boas práticas do <b>SOLID</b>.
A modelagem apresenta uma separação clara de responsabilidades entre as classes, promovendo baixo acoplamento,
alta coesão e facilidade de manutenção.

O sistema utiliza herança para especialização de usuários e itens, além de uma camada de serviço responsável
pelo gerenciamento dos empréstimos, garantindo organização, escalabilidade e extensibilidade do software.
</p>
